# shikamaku.simizu.3.84
飾磨区清水3-84の土地について
親から相続した耕作放棄地で、草刈り頻度も低いので、誰が所有者かわからないという感じで、所有者に成りすまして詐欺の話を持ち掛ける人がいたり、不法投棄する人がいたりするので、土地に立て札を立ててQRコードでこのページに飛ぶようにしました。

不動産の有効活用に興味が無いわけではないのですが、相続税対策用の利益率の賃貸住宅の提案が来ても無視します。他方で、店舗などでの利用は
４０m道路沿いではあるものの土地のサイズやすでに近隣に競合店がありなかなか難しいと思っています。
